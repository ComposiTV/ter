# ter

*(adjective)* The third iteration.

Coordinate term: [bis](https://github.com/SED4906/bis)

## Installation

Building an installation ISO can be accomplished on a Universal Blue system image with `just build && just build-iso`.

The administrator user should be given the username `ctv` when installing.

## How to Use

The compositor is niri, with a custom configuration. vi-style keybindings are not present, Mod+K toggles the on-screen keyboard.
